#  Очереди сообщений (RabbitMQ)

## Обзор системы сообщений

Quotation Book использует **RabbitMQ** для асинхронной обработки задач генерации файлов (png-изображений и pdf-файлов). Это позволяет не блокировать основной поток приложения при выполнении тяжелых операций.

## Очереди сообщений

### 1. `book_generation_queue`
- **Назначение**: Генерация обычных PDF книг
- **Приоритет**: Normal
- **Durable**: Да

### 2. `image_generation_queue`
- **Назначение**: Генерация изображений
- **Приоритет**: Normal
- **Durable**: Да

### 3. `large_book_generation_queue`
- **Назначение**: Генерация больших PDF книг
- **Приоритет**: High (для первых 2 задач)
- **Durable**: Да
