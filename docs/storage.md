# Хранилища данных

## PostgreSQL (Основная СУБД)

### Назначение:
- Хранение цитат в таблице `quotes`
- Управление пользователями(БД - Keycloak)

## Redis (Кэширование)

### Назначение:
- Кэширование userinfo от Keycloak (30 секунд по умолчанию)
- Кэширование цитат из PostgreSQL (10 секунд по умолчанию)
- Отслеживание статуса задач генерации

## Minio S3 (Хранилище файлов)

### Назначение:
- Хранение сгенерированных PDF книг в подготовленном bucket
- Хранение метаданных файлов

## NFS (Сетевая файловая система)

### Назначение:
- Совместный доступ к статическим файлам(только .png формат) между backend-сервисами(read write) и frontend(read only)
- Отдача статики через Nginx (/opt/app-root/static:ro)
